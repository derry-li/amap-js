# 介绍

AMapJS 是AMap高德地图API加载器。帮助开发者快速加载高德地图相关API，在模块化应用、异步编程中使用API更加灵活便捷。

# 特性

  • 支持 NPM 和 UMD 两种方式使用。
  • 支持指定 API 版本。
  • 支持插件模块加载。
  • 支持Promise。
  • 支持按需引入。
  • 允许多次执行加载操作，网络资源不会重复请求。

# 浏览器支持

支持现代浏览器和 IE9 及以上（需要 polyfills）。
