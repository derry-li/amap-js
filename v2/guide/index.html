# 介绍

AMapJS, 基于AMap高德地图构建的API加载器。

# 特性

  • 开箱即用的高质量 Loader。
  • 友好指定 API 版本。
  • 支持按需引入。
  • 支持插件模块加载。
  • 可以 Loader 队列加载。
  • 允许多次执行加载操作，网络资源不会重复请求。

# 浏览器支持

支持现代浏览器和 IE9 及以上（需要 polyfills）。
