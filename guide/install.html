# 安装

# 通过 npm 安装

# 通过 npm 安装
npm i amap-js -S

# 通过 yarn 安装
yarn add amap-js

# 使用 AMapJS

# 完整引入

import AMapJS from 'amap-js';

# 按需引入

import { AMapLoader } from 'amap-js';

完整列表

# 通过 CDN 引入

使用 AMapJS 最简单的方法是直接在 html 文件中引入 CDN 链接，之后你可以通过全局变量 AMapJS 访问到所有API。 目前可以通过unpkg.com/amap-js获取到最新版本的资源。

<script type="text/javascript" src="https://unpkg.com/amap-js/lib/amap-js.min.js"></script>

<script>
  console.log(AMapJS);
</script>

# 兼容性

AMapJS 支持所有的现代浏览器和 IE9+。

对于 IE 系列浏览器，需要提供 es5-shim 和 es6-shim 等 Polyfills 的支持。

如果你使用了 babel，强烈推荐使用 @babel/polyfill 和 @babel/plugin-transform-runtime 来替代以上两个 shim。

避免同时使用 babel 和 shim 两种兼容方法。
